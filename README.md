# Melon Loader Install

**Melon Loader Install** is a simple and reliable installer for **Melon Loader** — the popular universal mod loader for Unity-based games. Easily install Melon Loader for games like VRChat, Among Us, Gorilla Tag, and many others.

## Key Features
- Simple one-click installation of Melon Loader
- Automatic game detection
- Support for both IL2CPP and Mono Unity games
- Clean and user-friendly interface
- Easy uninstall option

## Getting Started
1. **Download** the latest version using the button above.
2. **Extract** the archive.
3. **Run** `MelonLoader.Installer.exe` as Administrator.
4. **Select your game** from the list.
5. Click **Install** and wait for the process to complete.
6. Launch your game — Melon Loader will load automatically.

**Pro Tip:** Place your mods in the `Mods` folder inside the game directory.
